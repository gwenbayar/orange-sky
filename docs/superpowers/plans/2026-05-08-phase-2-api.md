# Phase 2 — FastAPI Server (Plan)

**Goal:** A thin read-only FastAPI server that serves the snapshot built in
Phase 1, exposing the dashboard's data needs over HTTP/JSON.

**Tech:** Python 3.12, FastAPI, uvicorn. `pytest` + `httpx` for tests.
Snapshot path comes from `SNAPSHOT_DIR` env var, defaulting to
`data/snapshot/2020-labor-day`.

## Endpoints

| Method | Path                          | Source file in snapshot              |
| ------ | ----------------------------- | ------------------------------------ |
| GET    | `/api/health`                 | none — liveness                      |
| GET    | `/api/event`                  | `event.json`                         |
| GET    | `/api/fires`                  | `fires.json`                         |
| GET    | `/api/smoke-polygons`         | `smoke.json`                         |
| GET    | `/api/monitors`               | `monitors.json`                     |
| GET    | `/api/monitors/{id}/ts`       | `monitors_ts/<id>.json`              |
| GET    | `/api/historic/yearly`        | `historic.json` → `yearly` key       |
| GET    | `/api/historic/by-cause`      | `historic.json` → `by_cause` key     |

## Error contract

- snapshot file missing → `503` with `{"error": "Snapshot not built. ..."}`
- unknown monitor id → `404`
- unhandled → `500` (FastAPI default)

## Files

```
api/
├── __init__.py
├── main.py                # the app + endpoints + dependency for snapshot dir
└── tests/
    ├── __init__.py
    ├── conftest.py         # writes a tiny synthetic snapshot, sets SNAPSHOT_DIR
    └── test_endpoints.py   # one test per endpoint (happy + 503 + 404)
```

`pyproject.toml` gains `fastapi`, `uvicorn[standard]`, and `httpx` (test only).

## Tasks (executed inline, no subagent loop)

1. Add deps + create `api/` skeleton.
2. Write `api/main.py` with all 8 endpoints.
3. Write `api/tests/` with conftest + per-endpoint tests.
4. Run pytest. Smoke: `uvicorn api.main:app` and `curl /api/event` returns the
   real 2020 Labor Day snapshot manifest.
5. Commit per logical step (skeleton, app, tests).
